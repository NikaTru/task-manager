# Менеджер задач (Task Manager)

Учебное SPA-приложение для управления списком задач, разработанное в рамках практических работ по дисциплине «Вёрстка и создание приложений».

## 🌐 Онлайн-версия

Приложение доступно по ссылке: [https://nikatru.github.io/task-manager/](https://nikatru.github.io/task-manager/)

## 📋 Функциональность

- Добавление новых задач
- Удаление задач
- Отметка задач как выполненных (чекбокс)
- Прогресс-бар выполнения задач
- Загрузка задач из внешнего API (JSONPlaceholder)
- Навигация между страницами без перезагрузки (SPA)
- Адаптивный дизайн (корректное отображение на мобильных устройствах)

## 🛠 Используемые технологии

- **React 19** — JavaScript-библиотека для создания интерфейсов
- **React Router 6** — маршрутизация в SPA
- **Vite** — инструмент сборки
- **CSS** — стилизация (адаптивная вёрстка)
- **gh-pages** — публикация на GitHub Pages

## 🔌 Используемое API

Данные загружаются из [JSONPlaceholder](https://jsonplaceholder.typicode.com/) — бесплатного REST API для тестирования.

Эндпоинт: `https://jsonplaceholder.typicode.com/todos?_limit=10`

Структура данных задачи:

- `id` — уникальный идентификатор
- `title` — текст задачи
- `completed` — статус выполнения

## 🚀 Инструкция по запуску

### Требования

- Node.js (версия 18 или выше)
- npm

### Установка и запуск

# Клонирование репозитория

git clone https://github.com/nikatru/task-manager.git

# Переход в папку проекта

cd task-manager

# Установка зависимостей

npm install

# Запуск в режиме разработки

npm run dev

# Открыть в браузере

http://localhost:5173/

### Сборка для продакшена

# Создание оптимизированной сборки

npm run build

# Предпросмотр собранной версии

npm run preview

### Публикация на GitHub Pages

npm run deploy
